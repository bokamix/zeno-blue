# Demo Files

This folder contains example files for testing ZENO's capabilities.

## Files

### budzet-2024.xlsx
A messy company budget spreadsheet with:
- Multiple sheets (Main budget, Detailed expenses, Plan vs Reality)
- Intentional issues: duplicates, inconsistent naming, missing data
- Notes and comments to review
- Perfect for testing data cleanup and analysis

### raport-kwartalny-q3-2024.docx
Q3 2024 quarterly report for a fictional company "TechSolutions":
- Executive summary
- Financial results
- Product development updates
- Team information
- Challenges and recommendations

### faktury/ (folder)
4 PDF invoices from October 2024:
- FV/2024/0156 - ABC Consulting
- FV/2024/0157 - XYZ Manufacturing  
- FV/2024/0158 - Startup Dreams
- FV/2024/0159 - Global Trade
Total value: ~52,500 PLN

### tablica-sprint.png
An image of a sprint planning board/whiteboard with:
- Task list with assignees and estimates
- Status tracking (Done, In Progress, To Do)
- Summary box
- Handwritten notes and deadlines

### user-analytics.csv
User analytics data with 200 users:
- User ID, signup date, plan type
- Revenue, login activity, features used
- Support tickets, country
- Churn risk assessment
- Great for analyzing retention and identifying at-risk users
